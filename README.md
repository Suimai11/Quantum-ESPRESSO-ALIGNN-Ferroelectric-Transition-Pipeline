# Quantum-ESPRESSO-ALIGNN-Ferroelectric-Transition-Pipeline
A self-contained computational workflow for calculating ferroelectric phase transition barriers, combining ALIGNN-FF (GPU) for fast structure relaxation and Quantum ESPRESSO (GPU) for high-precision DFT refinement.  一款自包含的铁电相变能垒计算软件，结合 ALIGNN-FF (GPU) 快速弛豫和 Quantum ESPRESSO (GPU) 高精度 DFT 精算。
